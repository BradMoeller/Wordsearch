-------------------------------------------------------------------------------
Duolingo WordSearch Demonstration App
-------------------------------------------------------------------------------

To install the app from the terminal:
1) Open a terminal and navigate to the directory that contains this README
2) Type "./install.sh"

To run unit tests from the terminal:
1) Open a terminal and navigate to the directory that contains this README
2) Type "./gradlew test --daemon"


To install the app from Android Studio:
1) Open the WordSearch project in Android Studio
2) Select the "app" configuration
3) Press "run"

To run unit tests from the terminal:
1) Open the WordSearch project in Android Studio
2) In the Android project browser, drill into app->java
3) Right click on the androidTest and/or test directory, and select "run tests"
